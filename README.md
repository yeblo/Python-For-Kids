# Python-For-Kids
**Python Setup Instructions**
Step 1: Install Python
For Windows:

Go to https://python.org
Click "Download Python 3.12.x" (the yellow button)
Run the downloaded installer
IMPORTANT: Check "Add Python to PATH" at the bottom
Click "Install Now"
Wait for installation to complete
Click "Close"

For Mac:

Go to https://python.org
Download the macOS installer
Open the downloaded .pkg file
Follow the installation wizard
Complete installation

verify Python installation 
python --version

**VS Code Setup Instructions**
1. For All Operating Systems:
2. Go to https://code.visualstudio.com
3. Click "Download for [Your OS]"
4. Run the installer
5. Follow the setup wizard with default settings
6. Launch VS Code when installation completes


**Set Up VS Code for Python**
**Install Python Extension:**
  1. Open VS Code
  2. Click the Extensions icon (four squares) in the left sidebar
  3. Search for "Python"
  4. Find the official Python extension by Microsoft
  5. Click "Install"
